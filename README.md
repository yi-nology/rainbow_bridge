# 虹桥计划（Rainbow Bridge）

项目代号“虹桥计划（rainbow_bridge）”，灵感源自中国神话中连接人界与天界的七彩虹桥，寓意为前端团队搭建一座高效、安全的静态资源通道。

该项目提供一套自部署的静态资源中台，支持按项目 Key 管理资源的上传、浏览、导入导出和引用地址暴露。整体基于 CloudWeGo Hertz + GORM + SQLite/Mysql/PGSQL 的 Go 后端组成。

## 后续扩展建议

- 将 Token 校验接入公司现有的权限或密钥管理系统，支持定期轮换。
- 接入对象存储（如 AWS S3、阿里云 OSS）替换本地文件系统，提升可靠性。
- 为导入导出加入异步任务、进度通知与失败重试机制。
- 在前端接入更详尽的资源预览（图片缩略图、文本文件内嵌预览等）。
